# Анализатор зависимостей NuGet пакетов

## Описание
Программа для автоматического анализа предопределенных NuGet пакетов и получения информации об их прямых зависимостях.

## Особенности
- ✅ Автоматический анализ 8 популярных NuGet пакетов
- ✅ Не требует ввода данных от пользователя
- ✅ Использует только стандартные библиотеки Python
- ✅ Выводит подробные результаты в консоль

## Запуск
```bash
python main.py
